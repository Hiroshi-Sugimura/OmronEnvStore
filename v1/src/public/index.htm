HAL (Home-life Assessment List) 連携設定

クラウドサービスHALと連携し、生活状況を得点化して見える化します。 この機能を利用するにはHAL連携設定が必要です。 ConfigタブのHAL連携設定でAPIトークンを設定してください。

1. (HALでの作業)；HALにログインする。（外部ブラウザが開きます。）
2. (HALでの作業)；ユーティリティのProfileのページに行く。
3. (HALでの作業)；APIトークンをコピーする。
4. (HEMS-Loggerでの作業)；HAL連携設定に先ほどのAPIトークンを張り付ける。
5. (HEMS-Loggerでの作業)；登録ボタンを押す。

事前準備

動作させるためには電力会社のBルートサービスの利用申し込みが必要です。電力会社に申請をすると，Bルート認証ID設定とBルート認証パスワード設定を取得できます。
通信モジュールとしてWi-SUNモジュール（TESSERA RL7023，またはROHM BP35C2のどちらか）が必須です。
TESSERA RL7023: Web site
ROHM BP35C2: Web site
